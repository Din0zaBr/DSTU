# Робот с навигацией по маякам

Проект системы навигации робота на основе LoRa маяков.

## Структура проекта

```
Work/Robot/
├── Beacon.ino      # Код для маяка (передатчик GPS координат)
├── Rover.ino       # Код для робота (приемник + навигация)
├── EQUIPMENT.md    # Список необходимого оборудования
└── README.md       # Этот файл
```

## Описание

### Beacon.ino - Маяк
Устройство, которое:
- Получает GPS координаты
- Передает их через LoRa каждые 2 секунды
- Формат пакета: `"LABEL LAT LNG SPEED"`

### Rover.ino - Робот
Устройство, которое:
- Принимает сигналы от маяков через LoRa
- Вычисляет расстояние и направление до маяков
- Управляет моторами для движения к выбранному маяку
- Отображает информацию на OLED дисплее

## Быстрый старт

1. **Установите библиотеки Arduino:**
   - LoRa (by Sandeep Mistry)
   - TinyGPS++ (by Mikal Hart)
   - Adafruit GFX Library
   - Adafruit SSD1306

2. **Настройте параметры:**
   - В `Beacon.ino`: измените `deviceName` и частоту `BAND` при необходимости
   - В `Rover.ino`: проверьте пины моторов и частоту `BAND`

3. **Загрузите код:**
   - Сначала загрузите `Beacon.ino` на одно устройство ESP32
   - Затем загрузите `Rover.ino` на другое устройство ESP32

4. **Подключите оборудование:**
   - См. схему подключения в `EQUIPMENT.md`

## Управление роботом

- **Кнопка (одиночное нажатие)**: Выбор маяка для навигации
- **Кнопка (удержание)**: Включение/выключение навигации

## Требования

- Arduino IDE 1.8+ или PlatformIO
- ESP32 Board Support Package
- Все библиотеки из списка выше
