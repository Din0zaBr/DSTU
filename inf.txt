Пояснение по запуску VPN-сервера и клиента на Python на разных компьютерах и в разных сетях

1. Описание программы:
   - Серверная и клиентская части написаны на Python.
   - Для шифрования используется алгоритм AES (библиотека pycryptodome).
   - Клиент отправляет зашифрованные сообщения, сервер их принимает и расшифровывает.

2. Запуск в одной локальной сети:
   - На сервере запустите server.py.
   - В client.py укажите IP-адрес сервера (например, 192.168.1.10).
   - Оба компьютера должны быть подключены к одной сети (Wi-Fi или кабель).
   - Порт 5555 должен быть открыт на сервере (проверьте фаервол).

3. Запуск в разных сетях (через интернет):
   1) Пробросьте порт на роутере сервера:
      - В настройках роутера найдите раздел Port Forwarding (Переадресация портов).
      - Добавьте правило: внешний порт 5555 -> внутренний порт 5555 -> IP сервера в локальной сети.
   2) Узнайте внешний (публичный) IP-адрес сервера (например, через сайт 2ip.ru).
   3) В client.py укажите этот внешний IP-адрес.
   4) Проверьте, что фаервол и антивирус не блокируют порт 5555.
   5) Запустите server.py на сервере, затем client.py на клиенте.

4. Важные моменты:
   - Ключ шифрования (KEY) должен совпадать в server.py и client.py.
   - Если внешний IP сервера меняется (динамический IP), используйте сервисы типа No-IP или DynDNS.
   - Если проброс порта невозможен (например, ограничения провайдера), используйте сервисы типа ngrok для создания туннеля.
   - Не передавайте секретные данные через открытые сети без дополнительной защиты.

5. Установка зависимостей:
   - Установите библиотеку pycryptodome командой:
     pip install pycryptodome

6. Пример запуска:
   - На сервере: python server.py
   - На клиенте: python client.py

При возникновении вопросов по настройке сети, пробросу портов или работе программы — обращайтесь за помощью!
